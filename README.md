## 빠른 시작

- CDN 사용: `<script src="https://unpkg.com/vue-router@4"></script>`
- [CodeSandbox](https://codesandbox.io/s/vue-router-4-reproduction-hb9lh)에서 브라우저 플레이그라운드 사용
- 기존 Vue 프로젝트에 추가:

  ```bash
  npm install vue-router@4
  ```
