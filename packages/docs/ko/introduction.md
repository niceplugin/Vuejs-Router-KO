Vue Router는 [Vue.js](https://vuejs.org)의 공식 라우터입니다. Vue.js 코어와 깊이 통합되어 Vue.js로 싱글 페이지 애플리케이션을 쉽게 구축할 수 있게 합니다. 다음과 같은 기능이 포함됩니다:

- 중첩된 라우트 매핑
- 동적 라우팅
- 모듈식 컴포넌트 기반의 라우터 설정
- 라우트 매개변수, 쿼리, 와일드카드
- Vue.js 트랜지션 시스템으로 구동되는 뷰(view) 트랜지션 효과
- 세밀한 탐색 제어
- 자동으로 활성화 CSS 클래스를 가지는 링크
- HTML5 히스토리 모드 또는 해시 모드
- 사용자 정의 가능한 스크롤 동작
- URL에 대한 올바른 인코딩
