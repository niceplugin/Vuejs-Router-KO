Vue Router는 [Vue.js](https://vuejs.org)의 공식 라우터입니다. Vue.js 코어와 긴밀하게 통합되어 싱글 페이지 앱을 쉽게 만들 수 있습니다. 기능은 다음과 같습니다:

- 중첩된 라우트 매핑
- 동적 라우팅
- 모듈식 컴포넌트 기반의 라우터 설정
- 라우트에 파라미터, 쿼리, 와일드카드 적용
- Vue.js의 트랜지션 시스템을 사용한 트랜지션 효과
- 라우트 탐색을 세밀하게 컨트롤
- 링크에 자동으로 엑티브 CSS 클래스 적용
- HTML5 히스토리 모드 또는 해시 모드
- 커스텀 스크롤 동작
- URL에 대한 적절한 인코딩
